## Mini blog

Build on Laravel 10.x (locally i have php 8.1)

## Project setup

+ Clone repo
+ `composer install`
+ `npm install`
+ `cp .env.example .env`
+ `php artisan key:generate`
+ `php artisan migrate`
+ `php artisan serve`

## Requirements
* Place values for (DB_DATABASE, DB_USERNAME, DB_PASSWORD) in your .env file
